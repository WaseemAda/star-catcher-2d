## Overview
Star Catcher is a 2D arcade-style game developed using the Godot Engine.  
The player controls a character that catches falling stars to increase their score.

## Features
- Real-time star spawning system
- Collision detection using Godot physics
- Score tracking system
- Game over condition
- Start menu and restart functionality
- Background music and sound effects

## Technologies Used
- Godot Engine
- GDScript
- Scene-based architecture
- 2D physics and collision system

## Controls
Move Left: Left Arrow
Move Right: Right Arrow
